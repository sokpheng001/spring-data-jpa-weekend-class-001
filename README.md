This project will underline for Spring boot development of RESTful API with Spring Data JPA module for data accessing.
Project Topic: E-commerce RESTful API
Database Table Relationship: 
```
Categories (1) ↔ (N) Categories_Products or Products (One to One or Many Relationship )

Products (1) ↔ (N) Categories (One to One or Many Relationship)

Orders (1) ↔ (N) Products or OrderItems (One to One or Many Relationship)

Products (1) ↔ (N) OrderItems (One to One or Many Relationship)

Customers (1) ↔ (N) Orders (One to Zero or One or Many Relationship)
```
